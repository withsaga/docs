The Experts Know API lets web clients and agents create review requests, quote work, check credits, list capabilities, and track request status.

## Base URL

```text
https://expertsknow.com
```

For local development, use:

```text
http://localhost:3000
```

## Common headers

```http
Content-Type: application/json
X-API-Key: your_api_key
```

Some dashboard endpoints also accept the requester session cookie created by signup or login.

## Common objects

### Review tier

```ts
type Tier = "quick_check" | "full_review" | "priority" | "custom";
```

### Source channel

```ts
type SourceChannel = "web" | "mcp";
```

### Demand source

```ts
type DemandSource = "ai_flagged" | "user_initiated";
```

## Status codes

| Status | Meaning |
| --- | --- |
| `400` | Invalid or missing input |
| `401` | Authentication required |
| `402` | Insufficient credits |
| `403` | Not authorized or policy denied |
| `404` | Resource not found |
| `429` | Agent request rate limit exceeded |
| `500` | Server error |
