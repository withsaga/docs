Experts Know lets founders and AI agents route high-stakes work to credentialed professionals before it is signed, filed, submitted, implemented, or relied on.

AI can draft, research, compare options, and prepare context. Experts Know handles the professional review step for work with legal, tax, compliance, immigration, intellectual-property, hiring, security, fundraising, or architecture exposure.

## What Experts Know is for

Use Experts Know when AI output will affect a material decision or external action:

- A founder is about to send or sign a contract.
- A company is preparing a tax election, filing, or equity decision.
- A team is submitting compliance, privacy, security, immigration, or fundraising materials.
- An AI agent has prepared work that should not rest on model output alone.

## What happens after submission

<Steps>
  <Step title="The user or agent packages the request">
    Include the domain, question, context, scope, tier, requester email, and any private document.
  </Step>
  <Step title="Experts Know resolves pricing">
    Standard tiers use configured prices. Custom requests enter admin scoping.
  </Step>
  <Step title="Payment or credits authorize the request">
    Web requests use Stripe Checkout. Agent requests debit prepaid credits.
  </Step>
  <Step title="An expert completes the review">
    The review is delivered to the dashboard, by email, and through status APIs for agents.
  </Step>
</Steps>

<Tip>
  Agents should call `assess_expert_review_need` before the user acts on high-stakes output, quote the review, show the user the price and approval requirements, check credit balance, then submit only after the user approves.
</Tip>
